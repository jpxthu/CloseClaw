# 代码块渲染

## 概述

代码块渲染是 IM Adapter 模块的通用渲染子功能，负责将消息中的代码块（三反引号包裹的代码段）按语法高亮渲染。各平台插件复用此能力，按自身特性选择不同的高亮策略。

## 架构

代码块渲染集成在各平台 Renderer 中，按平台能力选择高亮策略。核心逻辑分为两步：识别代码段和按平台渲染。

**平台渲染策略**：

| 平台 | 渲染方式 | 说明 |
|------|---------|------|
| 飞书 | 原生 markdown 代码块 | 飞书消息卡片原生支持三反引号代码块，平台端自行高亮 |
| CLI | ANSI 转义序列 | 按语言注入颜色码，终端渲染语法高亮 |
| 其他 | 纯文本 | 保留代码块标记，无额外着色 |

语言标注从 markdown 代码块标记中提取（如 ` ```rust ` 表示 Rust 高亮），平台自行决定支持的语言列表。

## 数据流

```
Renderer 处理 Text 块
  ↓
逐行解析 Text 块内容
  ↓
遇到 ```lang 标记 → 开始代码段
  ├── 记录语言标注
  ├── 收集代码行
  └── 遇到关闭 ``` → 结束代码段
  ↓
按平台选择渲染方式
  ├── 飞书：将整个 ``` 代码块作为 markdown 元素输出
  ├── CLI：逐行注入 ANSI 颜色码后输出
  └── 回退：保留原始 ``` 标记
  ↓
继续后续文本渲染
```

## 模块关系

- **上游**：各平台 Renderer（处理 Text 块时调用代码块识别逻辑）
- **下游**：IMPlugin（接收渲染后的代码块输出并发送）
- **所属**：IM Adapter 模块的通用子功能
