# CloseClaw Developer Skill

> Guide for agents that develop code within CloseClaw.

## Overview
This skill helps agents understand how to develop, test, and maintain CloseClaw code.

## Quick Reference
| Task | Command | Tool |
|------|---------|------|
| Run tests | `cargo test` | git_ops |
| Build | `cargo build` | git_ops |
| Commit | `git commit -m "..."` | git_ops |
| Push | `git push` | git_ops |
